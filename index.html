Git

  • "git" must preceed every Git statement
  • "checkout" command moves the working branch to a new branch
  • "-b" command creates a new branch
  • "add" adds modifications in the current working branch to the staging area
  • "git status" checks the current working branch
  • "git checkout -b branch-name" creates and switches to a new branch
